# Unmelting

## 1. 프로젝트 개요

**Unmelting**은 “녹지 않는 소녀” 카드 한 장을 조작해, 매 턴 한 칸씩 다가오는 보상과 재앙의 카드를 판단하며 살아남는 **카드 레인 로그라이트** 게임이다.

플레이어는 일반적인 덱빌딩처럼 많은 카드를 손패로 운용하는 것이 아니라, **캐릭터 카드 한 장**을 중심으로 스테이지를 돌파한다. 스테이지 위의 카드들은 매 턴 플레이어를 향해 전진하며, 플레이어는 제한된 행동력 안에서 무엇을 처리하고 무엇을 감수할지 선택해야 한다.

---

## 2. 장르

**카드 레인 로그라이트 / 스테이지형 카드 생존 게임**

### 장르 정의

> 플레이어 캐릭터 카드 한 장을 성장시키며, 매 턴 한 칸씩 다가오는 카드판을 읽고 보상과 위협 사이에서 생존 루트를 만드는 로그라이트.

---

## 3. 한 줄 소개

> **녹지 않는 소녀 카드 한 장으로, 매 턴 한 칸씩 다가오는 보상과 재앙의 카드를 판단하며 살아남는 몽글몽글 다크판타지 로그라이트.**

---

## 4. 핵심 콘셉트

플레이어는 **“녹지 않는 소녀”**라는 캐릭터 카드 한 장을 조작한다.

스테이지에는 다음과 같은 카드들이 배치된다.

- 적 카드
- 보상 카드
- 난관 카드
- 저주 카드
- 이벤트 카드
- 상점 카드

이 카드들은 매 턴마다 플레이어 쪽으로 **한 칸씩 전진**한다.

플레이어는 제한된 행동력 안에서 다음을 판단해야 한다.

- 보상을 먹을지
- 적을 먼저 처리할지
- 함정을 피할지
- 위험을 감수하고 더 큰 보상을 노릴지
- 카드 위치를 조작해 위기를 넘길지

---

## 5. 기본 화면 구조

### 단일 레인 예시

```text
[먼 거리]     [중간 거리]     [근접 거리]     [플레이어]
  보상 카드     적 카드        함정 카드       소녀 카드
```

### 3레인 구조 예시

```text
위쪽 레인     [보상] → [적] → [빈칸] → [소녀]
중앙 레인     [함정] → [보상] → [적] → [소녀]
아래 레인     [저주] → [빈칸] → [상점] → [소녀]
```

---

## 6. 기본 턴 구조

1. 플레이어가 행동력을 사용해 행동한다.
2. 모든 카드가 한 칸씩 플레이어 쪽으로 전진한다.
3. 근접 칸에 도달한 카드는 충돌 효과를 발동한다.
4. 빈 공간에 새로운 카드가 생성된다.
5. 다음 턴으로 넘어간다.

---

## 7. 핵심 재미

### 7.1 다가오는 카드판을 읽는 재미

카드들이 매 턴 전진하기 때문에 플레이어는 계속 판단해야 한다.

> “저 카드를 지금 처리할까, 한 턴 더 기다릴까?”

카드의 위치, 타입, 도달 타이밍이 곧 전투의 퍼즐이 된다.

---

### 7.2 보상과 위험 사이의 선택

이 게임에서 보상은 전투가 끝난 뒤 주어지는 것이 아니라, 위험 카드와 함께 전진해온다.

핵심 감정은 다음과 같다.

> “저 보상 먹고 싶은데, 지금 먹으면 위험하다.”

플레이어는 매 턴 안전한 선택과 욕심나는 선택 사이에서 고민하게 된다.

---

### 7.3 캐릭터 카드 한 장을 키우는 재미

덱을 크게 늘리는 방식이 아니라, **소녀 카드 한 장 자체가 점점 변화하고 성장**한다.

예시 성장 요소는 다음과 같다.

| 요소 | 역할 |
|---|---|
| 양초 | 생존 자원 / 회복 / 강화 |
| 우표 | 패시브 강화 |
| 밀랍 | 강하지만 위험한 강화 |
| 기억 | 스테이지 보상 능력 |
| 저주 | 리스크가 있는 능력 변화 |

플레이어가 느껴야 하는 감각은 다음과 같다.

> “이번 판의 내 소녀 카드는 전판과 다르다.”

---

### 7.4 카드 위치를 조작하는 재미

이 게임의 카드는 손패가 아니라 **공간 위의 오브젝트**다. 따라서 카드 위치 조작이 핵심 전략이 된다.

| 행동 | 효과 |
|---|---|
| 밀치기 | 위험 카드를 뒤로 보낸다 |
| 끌어오기 | 먼 보상을 앞으로 당긴다 |
| 봉인하기 | 카드의 전진을 멈춘다 |
| 자리바꾸기 | 적과 보상의 위치를 교환한다 |
| 태우기 | 한 레인의 카드들을 제거한다 |

---

## 8. 세계관 분위기

**Unmelting**은 밝고 귀여운 동화가 아니라, 부드럽지만 어딘가 불길한 다크판타지다.

### 분위기 키워드

- 녹지 않는 소녀
- 양초와 밀랍
- 따뜻한 촛불
- 축축한 검은 숲
- 작은 카드 캐릭터
- 귀엽지만 기괴한 적들
- 조용히 다가오는 재앙
- 몽글몽글한 다크판타지

### 비주얼 톤

| 요소 | 방향 |
|---|---|
| 배경 | 짙은 남색, 먹색, 어두운 보라 |
| 주요 포인트 | 따뜻한 노랑, 양초빛 주황 |
| 위험 요소 | 검붉은색, 탁한 자주색 |
| UI | 낡은 종이색, 밀랍색, 검은 잉크 |
| 카드 테두리 | 타버린 종이, 밀랍 봉인, 촛농 |

---

## 9. 주인공

## 녹지 않는 소녀

양초로 이루어졌거나, 양초와 관련된 저주를 받은 존재. 평범한 양초처럼 녹아 사라져야 하지만, 이상하게도 그녀는 녹지 않는다.

그래서 세계는 그녀를 불길한 존재로 본다.

### 캐릭터 이미지

- 작은 소녀
- 촛불 또는 랜턴을 들고 있음
- 머리카락이나 옷자락에 밀랍 같은 질감
- 따뜻한 노란빛과 차가운 어둠의 대비
- 귀엽지만 무표정하거나 공허한 표정
- 종이 카드 위에 그려진 말 없는 인형 같은 인상

## 10. 카드 종류

| 카드 종류 | 역할 |
|---|---|
| 적 카드 | 도달하면 피해를 준다 |
| 보상 카드 | 도달하거나 획득하면 이득을 준다 |
| 난관 카드 | 제거하거나 피해야 한다 |
| 저주 카드 | 강한 효과와 부작용을 함께 준다 |
| 이벤트 카드 | 선택지를 제공한다 |
| 상점 카드 | 강화나 회복을 제공한다 |

### 11.1 보상 카드

| 카드 | 효과 |

| 진흙길 | 레인 이동 제한 |
| 찢어진 길 | 특정 카드 획득 불가 |

| 봉투 먹는 까마귀 | 보상 카드를 훔치는 적 |

| 꺼진 아이 | 침묵/저주 계열 적 |

## 16. MVP 전투 규칙

```text
3레인 x 4칸
```

