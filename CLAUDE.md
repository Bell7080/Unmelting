# CLAUDE.md

Unmelting 협업용 핵심 요약. **현재 코드 기준**으로만 유지한다.

## 프로젝트 요약
- 장르: 3레인 × 3행 카드 레인 로그라이트.
- 플레이: `녹지 않는 소녀` 1장 중심 + 손패/조합/유물.
- 톤: 따뜻한 촛불, 낡은 종이, 어두운 남보라/먹색 다크판타지.

## 실행/검증
```bash
npm install
npm run dev
npm run type-check
npm run test
npm run build
```

## 구현 우선 사실(2026-05-21 기준)
- 턴 루프: 플레이어 행동 → 적/보물/폭탄/꽃 처리 → 불씨/레일 하강·리필 → 포자 전염.
- 상점 루프: 10/20턴 상점, 30턴 제단 모드.
- 상점/제단 콘텐츠는 `shop-content-bundle` 레이어로 **동시 공개**.
- 제단: 리롤 없음, 무료 카드 2장(기본+수당), 팩 4종(축복/자원/강화/삭제).
- 상점: 리롤은 유물 슬롯만 교체(무료/팩 고정).
- 무료 카드 `선물 상자`: 방문마다 랜덤 효과 1개, 즉시 소모.
- 유물/팩/무료카드 희귀도는 `ShopPools` 공통 팔레트(common~legendary) 사용.

## UI/UX 규칙
- 이모지 신규 추가 금지. `src/ui/Icons.ts`의 플랫 SVG 스타일 유지.
- 촛불/밀랍/낡은 종이 테마 유지(색·테두리·그림자 일관성).
- 수치 피드백(불빛/화폐/체력/방패/불씨/게이지)은 트레일/버스트/롤링 카운트가 같은 beat에 보이게 유지.

## 코드 규칙
- TypeScript only, import 주변 try/catch 금지.
- 새 코드에는 의도 중심의 짧은 주석 포함.
- 테스트/더미/과거 실험 잔여 코드는 남기지 말고 제거 또는 보고.

## 문서 규칙
- 장문 패치노트 누적 금지.
- 사실/규칙/주의점만 유지하고 중복 설명은 제거.
